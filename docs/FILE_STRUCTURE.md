# File structure

## Top-level layout

- [README.md](../README.md): project overview, quick start, and documentation links.
- [AGENTS.md](../AGENTS.md): short agent-facing pointers to repo rules.
- [package.json](../package.json): npm scripts and TypeScript/browser development dependencies.
- [build_github_pages.sh](../build_github_pages.sh): production build into `dist/`.
- [run_web_server.sh](../run_web_server.sh): local preview server for the built `dist/`
  artifact.
- [check_codebase.sh](../check_codebase.sh): local codebase verification gate.
- [src/](../src/): browser game source.
- [tests/](../tests/): Python repo hygiene tests, Node game tests, and Playwright browser tests.
- [data/science_career_paths/](../data/science_career_paths/): source drafts for scientist-path
  content.
- `docs/`: project documentation and shared repo rules.

## Generated artifacts

- `dist/` is generated by `npm run build` and ignored by git.
- `test-results/` and `playwright-report/` are generated by Playwright and ignored by git.
- `node_modules/` is installed by npm and ignored by git.

## Documentation map

- [docs/INSTALL.md](INSTALL.md): dependency setup and install verification.
- [docs/USAGE.md](USAGE.md): play loop, content authoring, build, serve, and test commands.
- [docs/CODE_ARCHITECTURE.md](CODE_ARCHITECTURE.md): source modules and runtime data flow.
- [docs/FILE_STRUCTURE.md](FILE_STRUCTURE.md): repository layout and where new work belongs.
- [docs/FILE_FORMATS.md](FILE_FORMATS.md): runtime content and local-storage data surfaces.
- [docs/TROUBLESHOOTING.md](TROUBLESHOOTING.md): known local build and browser-test issues.

## Where to add new work

- Add game logic under [src/](../src/) with focused feature-owned modules.
- Add launch-path source material under [data/science_career_paths/](../data/science_career_paths/).
- Add fast deterministic tests under [tests](../tests/).
- Add browser interaction tests under [tests/playwright/](../tests/playwright/).
- Add user-facing docs under `docs/` with ALL CAPS filenames.
