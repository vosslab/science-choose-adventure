# Roadmap

This roadmap captures likely next work after the first playable GitHub Pages build.

## Near term

- Tune the neutral `CORE_DECK` and `FLAVOR_POOL` cards after playtesting the resemblance pilot.
- Add broader Playwright coverage for touch swipe behavior.
- Add visual screenshots for desktop and mobile review.
- Tune the four-C balance so hidden stat direction feels inferable rather than arbitrary.

## Later

- Expand the core and flavor card sets and grow the scientist roster beyond the five pilot
  scientists; the end result is a resemblance reveal, not ending types.
- Decide whether a framework such as Solid.js is useful if UI state grows beyond the one-card
  loop.
- Consider moving content authoring out of TypeScript only if deck editing becomes the bottleneck.

## Known gaps

- Collect player feedback before committing to larger roster or framework work.
